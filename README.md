# Dotfiles
A collection of configuration files for my personal setup.
based on the setup [here](https://github.com/antsouchlos/dotfiles) from antsouchlos
waybar inspired by [waybar-macos-sequoia](https://github.com/kamlendras/waybar-macos-sequoia)

## Usage

$ stow -S sway waybar zsh kitty
